GitHub and Virtual Machines



750 Points
Learning Objectives

These exercises introduce basic skills and technologies used in lean product development.

After completing this assignment you will be able to:

  1. Create a virtual machine using a Type 2 hypervisor
    1. configure a Microsoft Azure account
    2. choose and startup a Microsoft Server virtual machine
  2. Create and upload files to a GitHub repository
    1. create a local repository
    2. create a GitHub repository
    3. commit changes to your repository
Assignment Overview

The goal of this assignment is to familiarize you with lean startup basic technologies.
